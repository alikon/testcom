This is the source of Jobs, a Joomla task manager extension. Here you will find the component, some plugins of type Job, 2 CLI commands and 2 webservices entrypoint.

## Requirements
Joomla >= 4.0

MySQL >= 5.6.0 or PostgreSQL >= 11.0

PHP >= 7.2.5
